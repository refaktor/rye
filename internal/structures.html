Context related functions

Functions for handling and working with Context.

current-ctx

BFunction(0): Returns current context. (core) (current-ctx)

current-ctx |type?
; returns ctx

parent

BFunction(0): Returns parent context of the current context. (core) (parent)

cc:: context f: does parent |type?   , cc
; returns ctx

context

BFunction(1): Creates a new context with no parent (core) (context)

context x: does 1 + 1   ::cc , cc
; returns 2

isolate

BFunction(1): (core) (isolate)

isolate x: does t 1 + 1   t: ?try  ::cc , cc |type?
; returns error

raw-context

BFunction(1): (core) (raw-context)

cc:: raw-context name: Jim  |type?
; returns ctx

private

BFunction(1): Creates a new context with current parent, returns last value (core) (private)

private x: 3 , x + 30  ::x
; returns 33

extends

BFunction(2): Extends a context with a new context in place and returns it. (core) (extends)

c1: context x: 1  , c2: extends c1 y: 2  , c2
; returns 1

bind!

BFunction(2): (core) (bind!)

isolate x: does 1 + 1   |bind! current ::cc , cc
; returns 2

unbind!

BFunction(1): Accepts a Context and unbinds it from it's parent Context. (core) (unbind!)

context x: does t 1 + 1   t: ?try  |unbind! ::cc , cc |type?
; returns error
