Rye reference: Basics

Printing functions

Functions that print values to stdout.

print

BFunction(1): Prints a value and adds a newline. (core) (print)

print 123
; prints 123
;

prn

BFunction(1): Prints a value without newline. (core) (prn)

prn 123
; prints 123

prns

BFunction(1): Prints a value and adds a space. (core) (prns)

prns 123
; prints 123

printv

BFunction(2): Prints a value and adds a newline. (core) (printv)

printv 33 value is: {}
; prints value is: 33
;

probe

BFunction(1): Prints information about a value. (core) (probe)

probe 33
; prints Integer: 33]

Logic functions

Function that help with logical operations.

true

Pure BFunction(0): Returns a truthy value. (core) (true)

true
; returns 1

false

Pure BFunction(0): Returns a falsy value. (core) (false)

false
; returns 0

not

Pure BFunction(1): Turns a truthy value to a non-truthy and reverse. (core) (not)

not 0
; returns 1
not 1
; returns 0

and

Pure BFunction(2): Bitwise AND operation between two values. (core) (and)

1 .and 0
; returns 0
1 .and 1
; returns 1
0 .and 0
; returns 0

or

Pure BFunction(2): Bitwise OR operation between two values. (core) (or)

1 .or 0
; returns 1
1 .or 1
; returns 1
0 .or 0
; returns 0

xor

Pure BFunction(2): Bitwise XOR operation between two values. (core) (xor)

0 .xor 0
; returns 0
1 .xor 1
; returns 0
0 .xor 1
; returns 1
1 .xor 0
; returns 1

all

BFunction(1): Takes a block, if all values or expressions are truthy it returns the last one, otherwise false. (core) (all)

all 1 1 1
; returns 1
0 0 1  .all
; returns 0

any

BFunction(1): Takes a block, if any of the values or expressions are truthy, the it returns that one, in none false. (core) (any)

any 0 0 1
; returns 1
0 0 0  .any
; returns 0

is-zero 0

Working with strings

Functions that mostly work with strings.

capitalize

Pure BFunction(1): Takes a String and returns the same String, but with first character turned to upper case. (core) (capitalize)

capitalize anne
; returns Anne

to-lower

Pure BFunction(1): Takes a String and returns the same String, but with all characters turned to lower case. (core) (to-lower)

to-lower ANNE
; returns anne

to-upper

Pure BFunction(1): Takes a String and returns the same String, but with all characters turned to upper case. (core) (to-upper)

to-upper anne
; returns ANNE

join

Pure BFunction(1): Joins Block or list of values together. (core) (join)

join Marry Anne
; returns MarryAnne

join\with

Pure BFunction(2): Joins Block or list of values together. (core) (join\with)

join\with Marry Anne  ,
; returns Marry,Anne

split

Pure BFunction(2): Splits a string into a block of values using a separator (core) (split)

split Marry,Anne ,
; returns Marry Anne

split\quoted

Pure BFunction(3): Splits a line of string into values by separator by respecting quotes (core) (split\quoted)

split\quoted "Marry,Anne",Joe , "
; returns Marry,Anne Joe

split\every

Pure BFunction(2): Splits a string into a block of values using a separator (core) (split\every)

split\every abcbce 3
; returns abc bce

Many of functions that work with other collections also work with strings.

Conditional functions

Program flow control functions

if

Pure BFunction(2): Basic conditional. Takes a condition and a block of code. (core) (if)

if 1 101
; returns 101
if 0 101
; returns 0

otherwise

Pure BFunction(2): Conditional if not. Takes condition and a block of code. (core) (otherwise)

otherwise 1 101
; returns 0
otherwise 0 101
; returns 101

^if

Pure BFunction(2): Basic conditional with a Returning mechanism when true. Takes a condition and a block of code. (core) (^if)

^if 0 101
; returns 0

^otherwise

Pure BFunction(2): Basic conditional with a Returning mechanism when true. Takes a condition and a block of code. (core) (^otherwise)

^otherwise 1 101
; returns 0

either

Pure BFunction(3): The if/else conditional. Takes a value and true and false block of code. (core) (either)

either 1 101  202
; returns 101
either 0 101  202
; returns 202

switch

BFunction(2): Classic switch function. Takes a word and multiple possible values and block of code to do. (core) (switch)

switch 1 1 101  2 202
; returns 101
switch 2 1 101  2 202
; returns 202

cases

BFunction(2): Similar to Case function, but checks all the cases, even after a match. It combines the outputs. (core) (cases)

cases 1 1  + 10  1  + 100
; returns 111
cases 1 0  + 10  1  + 100
; returns 101

Looping functions

Program flow control functions

loop

Pure BFunction(2): Basic conditional. Takes a condition and a block of code. (core) (if)

loop 3 prns x
; prints x x x

for

Pure BFunction(2): Conditional if not. Takes condition and a block of code. (core) (otherwise)

for 1 2 3  prns x
; prints x x x
a b c  .for .prns
; prints a b c

forever

BFunction(1): Accepts a block and does it forever. (core) (forever)

forever once .prn .return
; prints once

forever\with

BFunction(2): (core) (forever\with)

forever\with 1 .prn .return
; prints 1

Doers and evaluators

Functions that do code and evaluate blocks.

do

Pure BFunction(1): Takes a block of code and does (runs) it. (core) (do)

do 101 + 101
; returns 202

do\in

BFunction(2): Takes a Context and a Block. It Does a block inside a given Context. (core) (do\in)

cc: context x: 101  do\in cc x + 101
; returns 202

with

BFunction(2): Takes a value and a block of code. It does the code with the value injected. (core) (with)

1 .with + 100
; returns 101

try

Pure BFunction(1): Takes a block of code and does (runs) it. (core) (try)

try 10 + unknown  |type?
; returns error

do\in\try

BFunction(2): Takes a Context and a Block. It Does a block inside a given Context. (core) (do\in\try)

ctx: context x: 101  do\in\try ctx x / 0  |type?
; returns error

vals

BFunction(1): Takes a block of Rye values and evaluates each value or expression. (core) (vals)

x: 101 vals x
; returns 101 
name: Anne vals name Goodall
; returns Anne Goodall 
y: 202 vals 101 y y + 101
; returns 101 202 303

vals\with

BFunction(2): Evaluate a block with injecting the first argument. (core) (vals\with)

vals\with 100 + 1
; returns 101

time-it

BFunction(1): Accepts a block, does it and times it's execution time. (core) (time-it)

time-it 1 + 1
; returns 0

Function creating functions

Functions that create functions.

does

Pure BFunction(1): Creates a function without arguments. (core) (does)

f: does prns Hey  f f
; prints Hey Hey

fn

Pure BFunction(2): Creates a function. (core) (fn)

f:: fn x  prns x  f A f B
; prints A B

fn1

Pure BFunction(1): Creates a function that accepts one anonymouse argument. (core) (fn1)

f:: fn1 .prns  f A f B
; prints A B

pfn

Pure BFunction(2): Creates a pure function. (core) (pfn)

add10: pfn x  x + 10  add10 9
; returns 19
x:: 10 f:: pfn  try prns 1   f |type?
; returns error
y:: 10 f:: pfn  try y   f |type?
; returns error

Pure functions only have access to other pure functions.

closure

Pure BFunction(2): Creates a function with specific context. (core) (closure)

x:: 10 cl:: fn  prns x  , context x: 99 cl
; prints 99
x:: 10 cl:: closure  prns x  , context x: 99 cl
; prints 10
