Rye reference: Misc

Type conversion and checking

Functions that convert between Rye value types.

to-word

Pure BFunction(1): Tries to change a Rye value to a word with same name. (core) (to-word)

to-word test
; returns test
to-word 'test
; returns test

to-integer

BFunction(1): Tries to change a Rye value (like string) to integer. (core) (to-integer)

to-integer 123
; returns 123

to-string

Pure BFunction(1): Tries to turn a Rye value to string. (core) (to-string)

to-string 123
; returns 123
to-string 'word
; returns word

to-uri

Pure BFunction(1): Tries to change Rye value to an URI. (core) (to-uri)

to-uri https://example.com
; returns https://example.com

to-file

Pure BFunction(1): Tries to change Rye value to a file. (core) (to-file)

to-file file.txt
; returns file://file.txt

to-context

BFunction(1): Takes a Dict and returns a Context with same names and values. (core) (to-context)

cc:: to-context dict name Jim  , cc
; returns Jim

is-integer

Pure BFunction(1): Returns true if value is an integer. (core) (is-integer)

is-integer 123
; returns 1
is-integer ABC
; returns 0

is-decimal

Pure BFunction(1): Returns true if value is a decimal. (core) (is-decimal)

is-decimal 123.456000
; returns 1
is-decimal 123
; returns 0
is-decimal ABC
; returns 0

is-number

Pure BFunction(1): Returns true if value is a number (integer or decimal). (core) (is-number)

is-number 123
; returns 1
is-number 123.456000
; returns 1
is-number ABC
; returns 0

is-string

Pure BFunction(1): Returns true if value is a string. (core) (is-string)

is-string ABC
; returns 1
is-string 123
; returns 0
is-string file://file.txt
; returns 0
is-string test@gmail.com
; returns 0

type?

Pure BFunction(1): Returns the type of Rye value as a word. (core) (type?)

type? ABC
; returns string
type? 123
; returns integer

Failure handling functions

Functions for handling and working with Context.

fail

BFunction(1): Constructs and Fails with an Error object. Accepts String as message, Integer as code, or block for multiple parameters. (core) (fail)

fail 404 |disarm |code?
; returns 404
fail message |disarm |mold
; returns Error: message 
fail 501 auth error  |disarm |mold
; returns Error(501): auth error

failure

BFunction(1): Constructs and Error object. Accepts String as message, Integer as code, or block for multiple parameters. (core) (failure)

failure 403 |type?
; returns error

code?

BFunction(1): Returns the status code of the Error. (core) (code?)

failure 403 |code?
; returns 403

disarm

BFunction(1): Disarms the Error. (core) (disarm)

failed?

BFunction(1): Constructs and Fails with an Error object. Accepts String as message, Integer as code, or block for multiple parameters. (core) (fail)

fail 403 |failed?
; returns 1

check

BFunction(1): Constructs and Fails with an Error object. Accepts String as message, Integer as code, or block for multiple parameters. (core) (fail)

1 / 0 |check 501 |code?
; returns 501

fix

BFunction(1): Constructs and Fails with an Error object. Accepts String as message, Integer as code, or block for multiple parameters. (core) (fail)

100 / 0 |fix 99  |+ 1
; returns 100
101 101 
100 / 1 |fix 99  |+ 1
; returns 101.000000

assert-equal

Pure BFunction(2): Test if two values are equal. Fail if not. (core) (assert-equal)

assert-equal 10 10
; returns 1
assert-equal 10 20 |disarm |type?
; returns error

Code flow combinators

Program flow control functions

pass

BFunction(2): Accepts a value and a block. It does the block, with value injected, and returns (passes on) the initial value. (core) (pass)

xxx .pass + yyy :xy
; returns xxx

keep

BFunction(3): Do the first block, then the second one but return the result of the first one. (core) (keep)

10 .keep + 1  * 10
; returns 11

wrap

BFunction(2): Accepts a value and a block. It does the block, with value injected, and returns (passes on) the initial value. (core) (wrap)

wrap 1 + 1  10 + 20
; returns 30

Functions that change values in-place

inc!

BFunction(1): Searches for a word and increments it's integer value in-place. (core) (inc!)

a:: 100 , inc! 'a
; returns 101
a:: 100 , inc! 'a , a
; returns 101

change!

BFunction(2): Searches for a word and changes it's value in-place. If value changes returns true otherwise false (core) (change!)

a:: 1 change! 2 'a
; returns 1
a:: 2 change! 2 'a
; returns 0

Functions that change values in-place are used more rarely in Rye and have ! at the end.

remove-last!

BFunction(1): Accepts Block and returns the next value and removes it from the Block. (core) (remove-last!)

b:: 1 2 3  , remove-last! 'b , b
; returns 1 2

append!

BFunction(2): Accepts Rye value and Tagword with a Block or String. Appends Rye value to Block/String in place, also returns it . (core) (append!)

b:: 1 2 3  , append! 4 'b , b
; returns 1 2 3 4 
b:: 1 2 3  , append! 4 5  'b , b
; returns 1 2 3 4 5  
b:: list 3 4  , append! list 1 2  'b , b
; returns L[ 3  4  1  2 ]
append! ab cd
; returns cdab
append! 3 4  1 2
; returns 1 2 3 4

change\ nth!

BFunction(3): Accepts a Block or List, Integer n and a value. Changes the n-th value in the Block in place. Also returns the new series. (core) (change\nth!)

b:: 1 4 3  , change\nth! b 2 4
; returns 1 4 3 
b:: 1 4 5  3  , change\nth! b 2 4 5
; returns 1 4 5  3 
b:: list 1 2 3  , change\nth! b 2 4
; returns L[ 1  4  3 ]
b:: list 1 2 3  , change\nth! b 2 list 4 5
; returns L[ 1  L[ 4  5 ]  3 ]
try b:: 1 2 3  , change\nth! b 4 0  |type?
; returns error
try b:: list 1 2 3  , change\nth! b 4 0  |type?
; returns error

sort!

BFunction(2): Accepts Rye value and Tagword with a Block or String. Appends Rye value to Block/String in place, also returns it . (core) (append!)

b:: 1 2 4 7  , sort! b , b
; returns 1 2 4 7

reverse!

BFunction(2): Accepts Rye value and Tagword with a Block or String. Appends Rye value to Block/String in place, also returns it . (core) (append!)

b:: 2 7 1 4  , reverse! b , b
; returns 2 7 1 4 
reverse! list 1 2 3
; returns L[ 3  2  1 ]
reverse! list 1
; returns L[ 1 ]
reverse! list
; returns L[]

Date and time functions

date

BFunction(1): Accepts a String and returns a Date object. (core) (date)

date 2023-01-01 |type?
; returns date
try date not-date  |type?
; returns error

datetime

BFunction(1): Accepts a String and returns a Date object. (core) (datetime)

datetime 2023-01-01T12:00:00 |type?
; returns time
try datetime not-time  |type?
; returns error

now

BFunction(0): Returns current Time. (core) (now)

now |type?
; returns time
